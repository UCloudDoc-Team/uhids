

# Agent管理

查看已安装和未安装Agent的主机信息。

![](/images/operation/agent管理.png)

表1.Agent列表参数

| 参数      | 说明                                                         |
| ------- | ---------------------------------------------------------- |
| 主机IP    | 主机的IP地址，默认显示内网IP，没有则显示外网IP                                 |
| 可用区     | 主机所在的可用区                                                   |
| Agent状态 | 共计2种状态：已安装和未安装。[安装agent参见此处](uhids/quick/agent)。 |
| 操作      | 点击【查看】后可跳转到uhost此主机所在的管理页面                                 |
