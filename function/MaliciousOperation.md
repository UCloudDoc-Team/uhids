

# 恶意操作检测 
文档更新时间（2025/11/05）
___

UHIDS主机入侵检测自主研发的恶意操作检测功能,拥有黑客恶意操作行为的检测手段, 并结合使用云端大数据分析和静态规则相结合的检测体系、分析恶意黑客在内网中进行横向攻击的操作手段,能够发现具有隐匿性的恶意操作行为。

## 检测原理

UHIDS主机入侵检测会实时监控主机系统上发生的进程行为，通过捕获进程启动路径、启动参数、所在环境以及进程的相关信息进行分析推理，将被判定为恶意操作的进程行为输出到恶意操作的事件中，触发告警提醒用户。

## 检测周期

- 插件启动后立即进行监控，已发生的行为无法感知，只针对新增事件行为进行监控。
- 风险在机器上消除后，界面的事件需要手动操作处理来置换事件状态。

## 检测项

| 检测项    | 说明                   |
| ------ | -------------------- |
| 恶意权限提升 | 支持检测在Linux机器上的恶意权限提升行为          |
